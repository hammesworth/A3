🔁 While

O laço while executa um bloco de código enquanto uma condição for verdadeira.

int i = 0;
while (i < 5) {
    cout << i << endl;
    i++;
}

Saída: 0 1 2 3 4

🔁 Do While

Diferente do while, o do while garante que o código será executado pelo menos uma vez.

int i = 0;
do {
    cout << i << endl;
    i++;
} while (i < 5);

Saída: 0 1 2 3 4

🔁 For

O for é ideal quando já se sabe o número de repetições. Compacto e direto.

for (int i = 0; i < 5; i++) {
    cout << i << endl;
}

Saída: 0 1 2 3 4

📦 Vetores (Arrays)

Um vetor armazena uma sequência de valores do mesmo tipo. O índice começa em 0.

int numeros[3] = {10, 20, 30};

for (int i = 0; i < 3; i++) {
    cout << numeros[i] << endl;
}

Saída: 10 20 30

Também é possível declarar o vetor sem inicializar:

int notas[5]; // valores indefinidos inicialmente

💻 Teste os exemplos em um compilador online, vulgo GDB.
